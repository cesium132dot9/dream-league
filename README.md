# Dream League
A gamified sleep schedule tracker application built with React/Vite. It has cute mascots which can be unlocked with good sleeping streaks and encouraging messages generated by Gemini-2.5 Flash. 

![image](/public/dreamleague-splash.png)

## Setup
1. Clone the repository
2. Install dependencies:
    ```bash
        npm install
    ```
3. Create a .env file from the template:
    ```bash 
        cp .env.example .env
    ```
4. Add your OpenRouter API key to ```.env```:
    ```bash 
        VITE_OPENROUTER_API_KEY=your_actual_api_key_here
    ```
5. Run the development server:
    ```bash 
        npm run dev
    ```
